# Kahve Otomasyon Sistemi ☕

Bu proje, C dili kullanılarak geliştirilmiş temel bir kahve satış otomasyonudur.

## Özellikler
* Renkli konsol arayüzü.
* Dinamik tutar hesaplama.
* Para üstü ve eksik ödeme kontrolü.
* Kullanıcı dostu çerçeveli menü yapısı.

## Nasıl Çalıştırılır?
1. Bir C derleyicisi (GCC, Dev-C++, Visual Studio) edinin.
2. `main.c` dosyasını derleyin ve çalıştırın.

## Kullanılan Teknolojiler
* **Dil:** C
* **Kütüphaneler:** `stdio.h`, `stdlib.h`, `conio.h`
